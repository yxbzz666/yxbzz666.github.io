Deploy

记录一下部署个站的过程

1. 环境新增

在云服务器上新建一个部署用户，并赋予sudo权限

创建名为 ‘XXX’ 的新用户（你可以换成其他名字）

sudo useradd -m -s /bin/bash XXX

设置用户密码

sudo passwd XXX

将用户添加到 sudo 组（如果需要 sudo 权限的话）

sudo usermod -aG sudo XXX

切换到 XXX 用户

su - XXX

验证当前用户

whoami
