Python装饰器详解

Python装饰器是Python中的一种高级功能，它允许你在不修改函数定义的情况下，向函数添加额外的功能，通常用于重复性的功能。
装饰器本质上是一个函数，它接受一个函数作为参数，并返回一个新的函数。
装饰器通常用于日志记录、权限检查、性能监控等场景。

面向切面编程（AOP），在函数执行前后添加额外的功能。
