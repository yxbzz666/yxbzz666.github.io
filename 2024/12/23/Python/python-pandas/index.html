python-pandas

数据类型

  • 一维数据：Series
  • 二维数据：DataFrame

数据的查看方法包括：

  • head()：查看前n行
  • tail()：查看后n行
  • info()：查看数据的基本信息，列索引、数据类型、非空计数和内存信息
  • describe()：查看数据的基本统计信息，包括计数、均值、标准差、最小值、四分位数、最大值
  • shape：查看数据的大小，行数和列数
  • dtypes：查看数据类型
  • columns：查看列名
  • index：查看行索引

Pandas数据IO需要额外的库支持：如openpyxl

  • read_csv()：读取CSV文件
  • read_excel()：读取Excel文件

数据库

python中可以使用sqlalchemy或者pymysql连接数据库。

  • 使用sqlalchemy连接数据库
  • 使用pymysql连接数据库

pandas中的read_sql()函数可以读取数据库中的数据。

to_sql()函数可以将数据写入数据库。

数据选取

loc和iloc是pandas中用于数据选取的两种方法。

  • loc：基于标签的选取，可以选取单行、单列、多行、多列，以及切片。

  • iloc：基于位置的选取，可以选取单行、单列、多行、多列，以及切片。

  • 布尔索引

数据集成

concat()函数可以用于数据集成，将多个数据集拼接在一起。

insert()函数可以用于数据集成，将一个数据集插入到另一个数据集的指定位置。

merge()函数可以用于数据集成，将两个数据集按照某个键进行合并。

数据清洗

重复数据过滤方法有：

  • drop_duplicates()：删除重复数据
  • duplicated()：返回一个布尔值，表示每个数据是否重复

空数据过滤方法有：

  • dropna()：删除空数据
  • fillna()：填充空数据
  • isnull()：返回一个布尔值，表示每个数据是否为空
  • notnull()：返回一个布尔值，表示每个数据是否不为空

删除行和列的方法有：

  • drop()：删除行或列，axis=0表示删除行，axis=1表示删除列。

filter()函数可以用于数据过滤，返回符合条件的行。
可以指定标签或使用正则表达式。

常见的数据清洗还有过滤异常值、数据转换、数据标准化等。

数据转换

轴索引修改方法有：

  • set_axis()：设置轴索引
  • rename()：重命名轴索引
  • rename_axis()：重命名轴索引

值替换方法有：

  • replace()：替换值
  • replace_na()：替换空值
  • replace_inf()：替换无穷大
  • replace_nan()：替换NaN

map()函数可以用于series,不能用于dataframe.

apply()函数可以用于dataframe,但是只能对行列进行操作,支持series，可操作多个series。

applymap()函数可以用于dataframe,对整个dataframe进行操作。

transform()函数可以用于dataframe,对整个dataframe进行操作,可以对多列进行不同操作。
